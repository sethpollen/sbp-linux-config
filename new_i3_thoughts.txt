PLAN

* [Alt       W         ] Cycle focus through windows in workspace
* [Alt       Tab       ] Cycle focus through workspaces in output

* [Alt       Esc       ] Jump to fresh workspace
* [Alt       digit     ] Jump to specified workspace

* [Alt       up/down   ] Cycle current workspace through outputs
* [Alt       left/right] Cycle current workspace through Z-order on output

* [Alt       S         ] Split layout
* [Alt       H         ] Horizontal split
* [Alt       V         ] Vertial split
* [Alt       T         ] Tabbed layout

Holding shift always grabs the current window:

* [Alt Shift Esc       ] Move window to fresh workspace
* [Alt Shift digit     ] Move window to specified workspace
* [Alt shift arrow     ] Move window within workspace
